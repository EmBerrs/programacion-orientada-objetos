# Practica_09 - Promedio, Media, Maximo, Minimo, Suma

## Objetivo
Ingresar 5 numeros y calcular promedio, media, maximo, minimo y suma.

## Diagrama de Clases

```
+---------------------+
| EstadisticasNumeros |
+---------------------+
| - numeros_[5]       |
| + ingresarNumeros() |
| + mostrarResultados |
+---------------------+
```

## Compilación

```bash
mkdir build && cd build && cmake .. && cmake --build .
./practica_09
```

# Practica_02 - Suma de 2 Números como Parámetros

## Objetivo
Implementar una clase que reciba dos números como parámetros y realice su suma.

## Descripción del Problema
Crear una clase que encapsule la operación de suma, recibiendo los operandos como argumentos de método.

## Diagrama de Clases

```
+-----------------+
|    Sumadora     |
+-----------------+
| + sumar(int)    |
| + sumar(double) |
+-----------------+
```

## Instrucciones de Compilación

```bash
cd Practica_02
mkdir build && cd build
cmake ..
cmake --build .
```

## Uso

```bash
./practica_02
```

Salida: Suma de 5 y 3: 8

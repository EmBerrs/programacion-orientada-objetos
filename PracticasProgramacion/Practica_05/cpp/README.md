# Practica_05 - Calculadora Nueva (Herencia)

## Objetivo
Crear CalculadoraNueva heredando de Calculadora base.

## Diagrama de Clases

```
+-------------+         +------------------+
| Calculadora |<--------| CalculadoraNueva |
+-------------+         +------------------+
```

## Compilación

```bash
mkdir build && cd build && cmake .. && cmake --build .
./practica_05
```

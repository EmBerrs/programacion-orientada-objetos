# Practica_03 - Calculadora

## Objetivo
Implementar una calculadora básica con las cuatro operaciones fundamentales.

## Descripción del Problema
Crear una clase Calculadora que ofrezca sumar, restar, multiplicar y dividir, recibiendo dos parámetros.

## Diagrama de Clases

```
+--------------------+
|    Calculadora     |
+--------------------+
| + sumar()          |
| + restar()         |
| + multiplicar()    |
| + dividir()        |
+--------------------+
```

## Instrucciones de Compilación

```bash
cd Practica_03 && mkdir build && cd build
cmake ..
cmake --build .
```

## Uso

```bash
./practica_03
```

# Practica_10 - Promedio con Arreglo

## Objetivo
Ingresar 5 numeros en una misma variable (arreglo) y calcular promedio, media, maximo, minimo, suma.

## Diagrama de Clases

```
+---------------------+
| EstadisticasArreglo |
+---------------------+
| - numeros_[5]       |
| + ingresarNumeros() |
| + mostrarResultados |
+---------------------+
```

## Compilación

```bash
mkdir build && cd build && cmake .. && cmake --build .
./practica_10
```

# Practica_24 - Archivos CSV

## Objetivo
Escribir y leer CSV con datos basicos, arreglos, tipos creados y ADT.

## Diagrama de Clases

```
+-------------+
| ManejadorCSV|
+-------------+
| escribir... |
| leerYMostrar|
+-------------+
```

## Compilación

```bash
mkdir build && cd build && cmake .. && cmake --build .
./practica_24
```

## Uso
Ejecuta desde el directorio build; crea archivos CSV en el directorio actual.

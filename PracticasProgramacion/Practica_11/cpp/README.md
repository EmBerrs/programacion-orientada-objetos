# Practica_11 - Promedio con Puntero al Arreglo

## Objetivo
Ingresar 5 numeros de manera indirecta (puntero) y calcular promedio, media, maximo, minimo, suma.

## Diagrama de Clases

```
+---------------------+
| EstadisticasPuntero |
+---------------------+
| - numeros_: double* |
| + ingresarNumeros() |
| + mostrarResultados |
+---------------------+
```

## Compilación

```bash
mkdir build && cd build && cmake .. && cmake --build .
./practica_11
```
